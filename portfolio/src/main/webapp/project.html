Track Star

 Kotlin, Android Studio, Firebase, Google Map API May 2022 - June 2022
A food and fitness android application that aims to track the progress of users' fitness journey. Main features include a navigation drawer with several fragments::
  • Home fragment: Users can set their goals and see their progress on their home page
  • Map fragment: utilizes Google Map API that tracks users' running/jogging routes by recording the location every 10 seconds.
  • List Fragment: shows the users' last 10 running paths on a map
  • Calorie fragment: shows the calorie burned and intake on that day and records the calorie consumption. The users can record their calorie consumption and refer to a chart that shows the calorie information of different food. The Calorie burned data comes from calculating the running speed and distance from map fragment.
  • Notification fragment: lets the users to set a reminder
  • Graph fragment: shows a line graph of calorie consumption and burned data of the last 7 days
